**ShopifyBackend Protocol** (runtime_checkable):
```python
class ShopifyBackend(Protocol):
    async def post_graphql(self, query: str, variables: dict | None = None) -> dict: ...
    async def post_graphql_mutation(self, query: str, variables: dict | None = None) -> dict: ...
    async def download_url(self, url: str) -> bytes: ...
    async def close(self) -> None: ...
```

**DirectBackend** class:
- Constructor takes `store: str`, `access_token: SecretStr`, `api_version: str = "2026-04"`.
- Creates `httpx.AsyncClient` with base_url `https://{store}/admin/api/{api_version}`, headers `X-Shopify-Access-Token` and `Content-Type: application/json`, timeout 60s connect 10s.
- `post_graphql()`: POST to `/graphql.json` with `{"query": query, "variables": variables or {}}`. Parse response. Log cost metadata from `extensions.cost.throttleStatus`. Check for THROTTLED errors and retry with backoff (extract the exact logic from `ShopifyClient._post_graphql` and `_handle_throttle` — lines 318-411 of shopify_client.py). Raise `ShopifyGraphQLError` on non-throttle errors, `ShopifyThrottledError` after max 3 retries.
- `post_graphql_mutation()`: Delegates to `post_graphql()` (same endpoint for Shopify — mutations go to the same GraphQL endpoint).
- `download_url(url)`: `httpx.AsyncClient().get(url)` and return `resp.content` (bytes). Use a fresh client since the URL is a signed external link (no auth headers).
- `close()`: `await self._client.aclose()`.

**CliBackend** class:
- Constructor takes `store: str`.
- `post_graphql()`: Calls `execute_graphql(store=self._store, query=query, variables=variables, allow_mutations=False)` from `shopify_exec.py`.
- `post_graphql_mutation()`: Same but `allow_mutations=True`.
- `download_url(url)`: Same as DirectBackend — fresh `httpx.AsyncClient().get(url)`, return bytes. (Signed URL, no Shopify auth needed.)
- `close()`: No-op (no persistent connection).

**create_backend(settings: Settings) -> ShopifyBackend** factory function:
1. If `settings.access_token is not None` (check for None, not empty string — SecretStr wraps the value): return `DirectBackend(store=settings.shop, access_token=settings.access_token, api_version=settings.api_version)`.
2. Elif `shutil.which("shopify") is not None`: return `CliBackend(store=settings.shop)`.
3. Else: raise `ShopifyCliNotFoundError` with message: "No Shopify credentials found. Either:\n  1. Install Shopify CLI and run: shopify store auth --store {settings.shop} --scopes read_orders,read_all_orders,read_products,read_inventory\n  2. Set SHOPIFY_FORECAST_ACCESS_TOKEN in .env"

Import `shutil` at module level. Import `execute_graphql` from `.shopify_exec`. Import `Settings` from `shopify_forecast_mcp.config`. Import exceptions from `.exceptions`.

**Create `src/shopify_forecast_mcp/core/shopify_exec.py`:**

```python
async def execute_graphql(
    store: str,
    query: str,
    variables: dict | None = None,
    allow_mutations: bool = False,
) -> dict:
```

Implementation:
- Build command list: `["shopify", "store", "execute", "--store", store, "--json", "--query", query]`
- If `variables` is not None and not empty: `cmd += ["--variables", json.dumps(variables)]`
- If `allow_mutations`: `cmd += ["--allow-mutations"]`
- Use `asyncio.create_subprocess_exec(*cmd, stdout=asyncio.subprocess.PIPE, stderr=asyncio.subprocess.PIPE)`
- `stdout, stderr = await proc.communicate()`
- If `proc.returncode != 0`: raise `ShopifyCliError(f"shopify store execute failed (exit {proc.returncode}): {stderr.decode().strip()}")`
- Parse `json.loads(stdout)`. If JSONDecodeError: raise `ShopifyCliError(f"Invalid JSON from shopify CLI: {stdout.decode()[:200]}")`
- Return the parsed dict.

Import `asyncio`, `json` at module level. Import `ShopifyCliError` from `.exceptions`.
