**Part A: ForecastResult dataclass** -- Create `src/shopify_forecast_mcp/core/forecast_result.py`:

```python
from dataclasses import dataclass, field
import numpy as np
```

`ForecastResult` dataclass with fields:
- `point_forecast: np.ndarray` -- shape `(horizon,)` for single series
- `quantile_forecast: np.ndarray | None` -- shape `(horizon, 10)`, channels `[mean, q10..q90]`
- `dates: list[str]` -- ISO date strings for each horizon step
- `confidence_bands: dict[str, np.ndarray]` -- pre-computed convenience: `{"q10": arr, "q20": arr, ..., "q90": arr, "mean": arr}`
- `metadata: dict` -- flexible: `{"metric": "revenue", "group": "store", "horizon": 30, "context_days": 365, "device": "cpu"}`

Class method `from_forecast(point, quantile, start_date, freq, metric, **meta) -> ForecastResult`:
- Takes raw `(batch, horizon)` point and `(batch, horizon, 10)` quantile arrays.
- Extracts single series (index 0) -- for batch, callers iterate and call per-series.
- Generates date strings: `pd.date_range(start=start_date, periods=horizon, freq=freq)`.
- Builds confidence_bands dict from quantile channels: `{"mean": quantile[0,:,0], "q10": quantile[0,:,1], ...}`.
- Returns `ForecastResult`.

`to_table(self, period: str = "weekly") -> str`:
- Groups horizon days into weekly (7-day) or monthly (30-day) buckets.
- For each bucket: sum point_forecast, compute mean q10 and q90.
- Returns markdown table:
  ```
  | Period | Projected | Low (10%) | High (90%) |
  |--------|-----------|-----------|------------|
  | Week 1 | $12,345   | $10,200   | $14,500    |
  ```
- Format values with commas and $ prefix if metric is "revenue" or "aov".

`summary(self, prior_period_value: float | None = None) -> str`:
- Total projected = sum of point_forecast.
- 90% CI = sum of q10 to sum of q90.
- If `prior_period_value` provided, compute trend: `((total - prior) / prior * 100)`.
- Return: "Revenue is projected to be $X over the next {N} days (90% CI: $Y-$Z). Trend: +N% vs prior period." or similar.
- If no prior_period_value: omit trend sentence.
