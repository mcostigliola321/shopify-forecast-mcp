  <action>
Add to `src/shopify_forecast_mcp/core/timeseries.py`:

1. `resample_series(series, freq, metric)`:
   - If freq == "D", return series unchanged (already daily).
   - Map metric to aggregator: "aov" -> "mean", all others -> "sum".
   - Use `series.resample(freq).agg(aggregator)`.
   - Handle edge case: empty series returns empty series.
   - Return the resampled series (DatetimeIndex is automatic from resample).

2. `clean_series(series, remove_outliers, outlier_method, interpolate_gaps)`:
   - If `interpolate_gaps` is True: `series = series.interpolate(method="linear")` then fill remaining edge NaN with 0.
   - If `remove_outliers` is False: return series as-is (after interpolation if requested).
   - If `outlier_method == "iqr"`:
     - Q1, Q3 = series.quantile(0.25), series.quantile(0.75)
     - IQR = Q3 - Q1
     - lower, upper = Q1 - 1.5 * IQR, Q3 + 1.5 * IQR
     - `series = series.clip(lower=lower, upper=upper)`
   - If `outlier_method == "zscore"`:
     - mean, std = series.mean(), series.std()
     - If std == 0, return series unchanged.
     - lower, upper = mean - 3 * std, mean + 3 * std
     - `series = series.clip(lower=lower, upper=upper)`
   - CRITICAL: Use `.clip()` not `.drop()` -- TimesFM needs continuous series.
   - assert len(result) == len(original) as a safety check.

3. Export both functions from the module.
