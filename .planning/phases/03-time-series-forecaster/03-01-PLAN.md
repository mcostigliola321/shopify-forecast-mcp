  <name>Task 2: Implement orders_to_daily_series to pass all tests</name>
  <files>src/shopify_forecast_mcp/core/timeseries.py</files>
  <behavior>
    - All 11+ tests from Task 1 pass
    - Zero-fill uses pd.date_range from min to max date, reindex with fill_value=0.0
    - AOV division uses safe division (0/0 = 0.0, not NaN)
    - Line-item-level aggregation for product/sku/collection grouping (not order-level)
    - Collection grouping fans out: one line item with product_id in 2 collections contributes to both
  </behavior>
  <action>
Implement `orders_to_daily_series` in `core/timeseries.py`:

1. Parse `local_date` strings into `pd.Timestamp` for each order.

2. For `group_by=None` (store-level):
   - Aggregate all line items across all orders per day:
     - "revenue": sum of `line_item["net_revenue"]` across all line items in all orders for that day
     - "orders": count of distinct orders per day (NOT line items)
     - "units": sum of `line_item["net_quantity"]` across all line items
     - "aov": revenue / orders with safe division (use `np.where(orders > 0, revenue / orders, 0.0)`)
   - Create pd.Series with DatetimeIndex, then reindex to full date_range (min to max date, freq="D"), fill_value=0.0.
   - Return `{"store": series}`.

3. For `group_by="product_id"` or `group_by="sku"`:
   - Iterate line items, group by the specified field.
   - For each group value, aggregate daily values same as store-level but only for that group's line items.
   - For "orders" metric with grouping: count orders that CONTAIN at least one line item with that group value.
   - Zero-fill each series independently across the full date range.
   - Return `{group_value: series}`.

4. For `group_by="collection_id"`:
   - Require `product_collection_map` parameter (raise ValueError if None).
   - For each line item, look up its `product_id` in the map to get collection_ids.
   - Fan out: if a product is in collections ["C1", "C2"], that line item contributes to both.
   - Products not in the map are skipped (logged as warning).
   - Aggregate and zero-fill per collection_id.

5. Export `orders_to_daily_series`, `Metric`, `GroupBy` from the module.

Run `uv run pytest tests/test_timeseries.py -v` -- all GREEN.
